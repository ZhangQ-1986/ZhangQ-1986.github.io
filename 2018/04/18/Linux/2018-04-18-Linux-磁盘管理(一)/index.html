Linux-磁盘管理(一)

发表于 2018-04-18 | 分类于 Linux

DF

df 查看已挂载磁盘的总容量、使用容量、剩余容量等，可以不加任何参数，默认是按k为单位显示。








DU

Linux du命令也是查看使用空间的，但是与df命令不同的是Linux du命令是对文件和目录磁盘使用的空间的查看，还是和df命令有一些区别的。


FDISK

Linux fdisk是一个创建和维护分区表的程序。只能划分2T以内的硬盘。

PARTED

当磁盘空间大于2T时，fdsik指令就不能满足要求。有两个方法来解决这个问题：其一是通过卷管理来实现，其二就是通过Parted工具来实现对2T磁盘进行分区操作。下面介绍parted。

MOUNT

mount挂载磁盘

mount [-t 文件系统] [-L Label名] [-o 额外选项] [-n] 装置文件名 挂载点

UMOUNT

umount [-fn] 装置文件名或挂载点

自动挂载

etc/fstab

要挂载的分区 挂载点 文件系统 挂载选项 是否备份 开始是否检查文件系统
/dev/sr0 /mnt/cdrom iso9660 defaults 0 0
