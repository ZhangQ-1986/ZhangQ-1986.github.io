CentOS7部署DVWA环境

发表于 2018-07-30 | 分类于 DVWA

DVWA下载

DVWA 的官方网站：http://www.dvwa.co.uk/
DVWA 的Github页面：https://github.com/ethicalhack3r/DVWA

CentOS7使用命令 wget https://github.com/ethicalhack3r/DVWA/archive/master.zip 来下载 DVWA。

如有提示bash: apt: command not found…的，请使用 yum install wget 命令安装 wget 工具。

使用 unzip DVWA-master.zip 来获得 DVWA 文件夹。

将之前解压出的 DVWA-master 文件夹放入 /var/www/html/
